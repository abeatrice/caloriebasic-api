# Calorie Basic API
This is the api for the caloriebasic mobile first web application.

#### This API supports:
	Registering a new user
	Logging In
	Logging Out
	Logging Out All Devices
	Get User
	Create Calories
	Get Calories

#### Postman Collection
	Open Postman
	Select Import
	Select File: caloriebasic.postman_collection.json
